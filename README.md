# Void Walker - Echoes of the Forgotten Kingdom

IB Personal Project (2025-2026) by Shamik Shikhar Murmu

A 2D adventure web game built with React and HTML5 Canvas.
Explore a forgotten kingdom, fight creatures, and uncover the story.

![Game preview](https://github.com/shamik-murmu/void-walker/blob/20b2ac9d6221275d8be17a47cd59154e3d3804e3/screenshot.png)

## Play Online
- Link: https://shamik-murmu.github.io/void-walker/

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Notes
- Controls are at the top left!
